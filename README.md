Udacity *CIFAR-10 Image Classifier* project: PyTorch CNN on CIFAR-10 to classify objects and compare build vs buy against Detectocorp.

Setup with [uv](https://docs.astral.sh/uv/): `uv venv && source venv/bin/activate && uv pip install -r requirements.txt`

Then start JupyterLab with: `jupyter lab` and open `CIFAR-10_Image_Classifier-STARTER.ipynb`. Run all cells; CIFAR-10 downloads automatically on first run.
